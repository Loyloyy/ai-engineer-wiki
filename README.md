# AI Engineer Wiki

A living knowledge base distilled from AI Engineer conference talks and GenAI practitioner material. Built on Andrej Karpathy's LLM Wiki pattern (April 2026): raw source transcripts feed into flat, entity-centric markdown pages — one page per concrete concept, pattern, tool, or company — each accumulating opinions and cross-links as new material arrives.

The wiki is written and maintained by [Claude Code](https://claude.ai/code) under a strict operational schema ([CLAUDE.md](CLAUDE.md)), with curation by Aloysius Lim. Every opinion claim is attributed to a specific speaker, talk, and timestamp. The compounding artifact is `wiki/` — a densely cross-linked reference that gets more useful with each new transcript ingested.
